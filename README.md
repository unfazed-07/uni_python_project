# 📊 Student Performance Analysis Dashboard

A data-driven dashboard built using **Python** and **Streamlit** to analyze and visualize student academic performance.

This project helps in understanding student trends, comparing performance, and generating insights from structured datasets.

[View Live App](https://insightdashboard.streamlit.app/)
---

## 🚀 Features

- 📌 Student-wise performance analysis  
- 📈 Semester-wise average comparison  
- 🏆 Best performing semester detection  
- 📊 KPI cards for quick insights  
- 📥 Export report functionality (PDF)  
- 🎯 Interactive dashboard using Streamlit  

---

## 🛠️ Tech Stack

- Python  
- Streamlit  
- Pandas  
- Matplotlib / Plotly  
- PDF Report Generation
